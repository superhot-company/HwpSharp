HwpSharp
========

Cross platform C# Library for the HWP document.


Requirement
-----------

- .NET 5.0


Build
-----

    git clone https://github.com/superhot-company/HwpSharp.git
    cd HwpSharp
    dotnet publish -c Release HwpSharp

Binaries could be found in `HwpSharp/bin`.
